# 🎧 Spotify Data Analysis Dashboard

🚀 An interactive Spotify-inspired analytics dashboard built using Power BI, Python, and SQL to uncover music trends, artist performance, and user engagement insights.

---

## 📌 Project Overview

This project transforms raw Spotify data into meaningful insights through an interactive and visually engaging dashboard.

It focuses on:
- Understanding song popularity trends
- Analyzing artist performance
- Comparing album vs single releases
- Identifying user engagement patterns

---

## 🎯 Key Features

✨ Spotify-inspired UI design  
📊 789 Songs & 342 Artists analyzed  
🎯 Image-based slicer using album covers  
📈 Monthly & yearly trend analysis  
📉 Artist-wise performance tracking  
⚡ Interactive filtering across multiple pages

---

## 🧠 Key Insights

- A small number of songs contribute to most of the popularity (Pareto principle)
- Certain artists consistently dominate engagement
- Seasonal trends impact song popularity
- Album and single releases show different audience reach patterns

---

## 🛠️ Tech Stack

- **Power BI** → Dashboard & Data Visualization  
- **SQL/MySQL** → Data Cleaning & Querying  
- **Python (Pandas, NumPy)** → Data Processing & EDA
